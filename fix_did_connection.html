🔧 Solucionar Error de Conexión D-ID

Problema: D-ID no permite conexiones desde localhost por seguridad. Necesitas usar un dominio público.

Opción 1: Usar ngrok (Recomendado)

ngrok crea un túnel público hacia tu localhost.

Paso 1: Instalar ngrok

Descarga e instala ngrok desde: https://ngrok.com/download

Paso 2: Iniciar túnel

ngrok http 8080

Paso 3: Copiar URL pública

ngrok te dará una URL como: https://abc123.ngrok.io

Paso 4: Configurar D-ID

Ve a D-ID Studio y agrega la URL de ngrok a los dominios permitidos.

Opción 2: Usar un servicio de hosting gratuito

Sube tu proyecto a un servicio como:

  • Vercel: https://vercel.com
  • Netlify: https://netlify.com
  • GitHub Pages: https://pages.github.com

Opción 3: Configurar D-ID para localhost (Avanzado)

Si tienes acceso a la configuración avanzada de D-ID:

  1. Ve a D-ID Studio
  2. Selecciona tu agente
  3. Ve a configuración/embed
  4. Agrega a dominios permitidos:
    http://localhost:8080
    http://127.0.0.1:8080
