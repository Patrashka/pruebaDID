🔧 Solución Alternativa - Fortigate Bloqueado

Problema: Fortigate (firewall corporativo) está bloqueando ngrok. Necesitamos una solución alternativa.

Opción 1: Usar un servicio de hosting gratuito (Recomendado)

Sube tu proyecto a un servicio gratuito que funcione con D-ID:

Vercel (Más fácil):

  1. Ve a https://vercel.com
  2. Crea una cuenta gratuita
  3. Conecta tu repositorio de GitHub
  4. Sube los archivos del proyecto
  5. Obtén la URL pública (ej: https://tu-proyecto.vercel.app)

Netlify:

  1. Ve a https://netlify.com
  2. Arrastra y suelta la carpeta del proyecto
  3. Obtén la URL pública

Opción 2: Usar tu IP pública

Si tienes acceso a tu router, puedes configurar port forwarding:

  1. Obtén tu IP pública: https://whatismyipaddress.com
  2. Configura port forwarding en tu router (puerto 8080)
  3. Usa tu IP pública en lugar de localhost

Opción 3: Usar un VPS gratuito

Servicios que ofrecen VPS gratuitos:

  • Oracle Cloud: https://cloud.oracle.com
  • Google Cloud: https://cloud.google.com
  • Railway: https://railway.app

Opción 4: Configurar D-ID para localhost (Si es posible)

Algunos usuarios reportan que esto funciona:

  1. Ve a D-ID Studio
  2. Selecciona tu agente
  3. Ve a configuración/embed
  4. Agrega estos dominios:
    http://localhost:8080
    http://127.0.0.1:8080
    http://0.0.0.0:8080

Opción 5: Usar un proxy online

Servicios que pueden ayudar:

  • localtunnel: npm install -g localtunnel
  • serveo: ssh -R 80:localhost:8080 serveo.net
  • cloudflared: cloudflared tunnel --url http://localhost:8080
💡 Recomendación: La opción más rápida es usar Vercel. Solo necesitas:
  1. Crear cuenta en Vercel
  2. Subir los archivos del proyecto
  3. Obtener la URL pública
  4. Agregar la URL a D-ID Studio
